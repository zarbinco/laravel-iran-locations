# Data Sources

The current manifest source is `excel-import` with source version `excel-initial`. The data has been normalized and shaped for this package's public schema, including stable codes, lifecycle fields, normalized/searchable names, and manifest counts/checksums.

This package does not claim the dataset is automatically complete, official, current, or suitable for legal, regulatory, logistics, or other high-stakes use. A stable public release should wait until source assumptions, completeness, and redistribution suitability have been verified.
